Les Boutons et Menus de WxPic

Les boutons et les menus permettent d'entrer les commandes de WxPic. Les boutons permettent d'effectuer les opérations les plus fréquentes. Les menus regroupent la quasi-totalité des fonctions disponibles.

La barre de boutons

Les boutons et leurs fonctions sont les suivants :

  • Ouvre un fichier HEX à choisir et charge son contenu en mémoire.
  • Écrit au format HEX le contenu de la mémoire dans un fichier à définir.
  • Recharge la mémoire avec le dernier fichier HEX chargé et programme le composant avec ce contenu.
  • Programme le composant avec le contenu de la mémoire.
  • Vérifie que le contenu de la mémoire du composant est identique à celui de la mémoire de WxPic. En cas de différence les mots de code et de données en erreur sont affichés en rouge dans les pages de code et de données.
  • Lit le contenu de la mémoire du PIC et l'enregistre dans la mémoire de WxPic.

 Le dernier élément de la barre est le commutateur rapide de session. Il permet de passer rapidement d'une session à une autre. La session choisie doit être Disponible pour que l'opération réussise sinon une erreur sera diagnostiquée et le changement de session n'aura pas lieu. Avant le changement de session, si la configuration d'origine avait été modifiée, elle est automatiquement sauvegardée. Utiliser le Dialogue de gestion des sessions si vous ne désirez pas sauvegarder les modifications de la configuration.

  • Fichier

  • Aide

Le menu Fichier

Le menu Fichier fournit les fonctions liés aux fichiers HEX et à la sortie de WxPic :

Charger un fichier HEX...:
Ouvre un fichier HEX à choisir et charge son contenu en mémoire.
Charger et Programmer... :
Ouvre un fichier HEX à choisir, charge son contenu en mémoire et programme le composant avec.
Recharger et Programmer :
Relit le fichier HEX précédemment ouvert, met à jour la mémoire avec son contenu et programme le composant avec.
Fichiers Récents :
Affiche les nom des 6 fichiers HEX les plus récemment chargés. Le choix d'un de ces fichiers recharge sont contenu en mémoire.
Écrire la mémoire dans un fichier HEX... :
Écrit au format HEX le contenu de la mémoire dans un fichier à définir.
Gérer les Sessions...
Ouvre le Dialogue de gestion des sessions.
Quitter sans sauver les paramètres :
Ferme WxPic sans enregistrer les modifications de configuration effectuées depuis l'ouverture du programme.
Quitter  :
Ferme WxPic après avoir enregistré les modifications de paramètres tels que le fichier HEX chargé, le programmateur et le composant utilisé...

Le menu Édition

Le menu Édition fournit les fonctions liées à la modification de la mémoire :

Effacer la mémoire
Efface le code, les données et la configuration chargée en mémoire.
Autoriser l'édition mémoire
Permet de modifier (lorsque ce choix est coché) la représentation hexadécimale de la mémoire code ou données. La modification doit être validée pour modifier effectivement la mémoire. La mémoire de configuration est toujours éditable.
Valider les modifications Mémoire
Applique à la mémoire, les modifications entrées dans les pages affichant la mémoire (Code et Données). La validation des modifications de la mémoire de configuration est spécifique.
Annuler les modifications Mémoire
Efface les modifications entrées dans les pages affichant la mémoire en réaffichant le contenu original de celle-ci.

Le menu Composant

Le menu Composant fournit les opérations sur le composant :

Sélectionner
Affiche l'onglet Config PIC et donne le focus à la liste de choix du type de composant.
Programmer
Programme le composant avec le contenu de la mémoire.
Programmer la config seulement
Programme uniquement le ou les mots de configuration.
Programmer l'Identification seulement
Programme les mots contenant l'identification utilisateur
Effacer
Efface le composant.
Vérifier
Vérifie que le contenu de la mémoire du composant est identique à celui de la mémoire de WxPic. En cas de différence les mots de code et de données en erreur sont affichés en rouge dans les pages de code et de données.
Vérifier l'effacement
Vérifie que le contenu du composant est bien vide comme il devrait l'être après effacement.
Lire
Lit le contenu de la mémoire du PIC et l'enregistre dans la mémoire de WxPic.
Réinitialiser
Réinitialise le programmateur et interrompt une éventuelle programmation par lot en cours.

Le menu Outils

Le menu Outil fournit des fonctions utilitaires diverses :

Démarrer Programmation d'un lot
Démarre une programmation d'une série de composants. Entre chaque composant WxPic attend une confirmation ou l'appuis sur le bouton du programmateur.
Sauvegarder la liste des composants prédéfinis...
Écrit dans un fichier à définir, la liste des composants reconnus par construction par WxPic.
Afficher les Options du programmateur
Affiche l'onglet Options.
